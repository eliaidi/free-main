free-main
=========
free-main是个人用于学习开发的应用网站,暂时功能只有blog
基于thrift协议服务形式开发...
下面是项目所有的架构

- 语言
  - java

- 显示端框架
  - html5
  - js
  - css3
  - bootstrap 2.2
  - ajax
  

- 前端框架:
  - spring 3.1.2
	- springmvc 3.1.2
	- freemarker 2.3.19
	- logback 1.0.7
	- javax.servlet 2.5

- 中间件框架:
  	- thrift 0.9.0
	- jackson 1.9.11
	- junit 4.10

- 后端框架:
	- spring 3.1.2
	- spring data redis 1.0.2.RELEASE
	- spring data mongodb 1.1.1.RELEASE
	- logback 1.0.7

- 通用服务
	- elasticsearch 0.20.2 -->> 基于lucene

- 缓存:
	- redis 2.6.12

- 数据库:
	- mongodb 2.4
	
- 服务器
  - nginx 1.2.8
  - jetty 8.5

- 开发工具
  - eclipse 3.6

- 代码管理
  - git
  - maven
  
- 项目进度管理
  - remine-1.4.1
  
- 代码质量监控
  - sonar
